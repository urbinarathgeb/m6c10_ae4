# Módulo 6 - Aprendizaje Esperado nº4

## Renderizado con HBS

### Objetivo

Implementar motor de plantilla HBS en Express

### Descripción

Implementar el motor de plantillas HBS en Express, creando rutas y vistas dinámicas en formato `.hbs`.

### Instrucciones

- Utilizar el proyecto Github visto en clases.
- Agregar 2 vistas nuevas, aparte de `Home` y `About`en el archivo `app`. Cada vista debe tener al menos 2 parámetros.
- Las vistas deben estar en formato `.hbs`.
- No es importante enfocarse en el diseño Front Ends.
